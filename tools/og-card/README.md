# OG social-card generator ("Aurora Glass" variant)

Generates the Open Graph / Twitter share cards in `/og/` and the per-language
landing pages in `/<lang>/`, from real in-game screenshots.

## Files
- `template-en.html` — the design reference (English), pixel source of truth.
- `l10n-v.mjs` — per-language card copy (kicker, title, subhead, pills, CTA) + `og:locale`.
- `cdp.mjs` — tiny headless-Chrome driver over the DevTools Protocol (no deps).
- `capture-hud.mjs` — drives the app into a planing state and screenshots the HUD
  (desktop + mobile) per language → `plates/` (git-ignored).
- `render-cards.mjs` — composites each card (`template` + plates) and writes
  `/og/og-cover[-<lang>].png` (1200×630, rendered @2× then downscaled).
- `pages.mjs` — writes `/<lang>/index.html` (localized meta + hreflang), injects
  hreflang into the root, and rebuilds `/sitemap.xml`.
- `plates/` — captured screenshots (git-ignored, regenerated by `capture-hud.mjs`).
  The shared backdrop `plates/plate-scene-en.png` is language-independent
  (captured once with the whole `#sidebar` HUD hidden).

## Regenerate (from repo root)
```bash
NODE=/Users/viachm/.nvm/versions/node/v22.15.0/bin/node   # nvm in non-interactive shell
python3 serve.py 8752 &                                    # serve the app locally

# 1) capture localized HUD screenshots (add/limit langs as needed)
$NODE tools/og-card/capture-hud.mjs en uk de fr es it pt nl pl cs sv da fi el tr id ru zh ja ko

# 2) render the cards → /og/
$NODE tools/og-card/render-cards.mjs        # all langs; en → og/og-cover.png

# 3) rebuild localized pages + hreflang + sitemap
$NODE tools/og-card/pages.mjs 2026-07-10    # arg = <lastmod> date
```

## Notes
- Card copy keeps the marketing "free" CTA; page **meta** stays free-less.
- On any image change, bump the `?v=N` cache-buster in `pages.mjs` (img) **and**
  in the root `index.html` og:image/twitter:image/structured-data, so social
  scrapers re-fetch. Then re-scrape via each platform's debugger.
- Auto-fit: the title shrinks (nowrap) and pills wrap so longer languages never
  collide with the phone — devices are never moved per language.
